# Mundial Familia PUSDÁ 2026

Página familiar mundialista para votar por los partidos de eliminación directa.

## Archivos
- index.html
- styles.css
- app.js
- config.js
- SUPABASE_SQL.txt

## Configuración
1. Crear proyecto gratis en Supabase.
2. Ejecutar el SQL de SUPABASE_SQL.txt.
3. Copiar Project URL y anon/public key.
4. Pegar esos datos en config.js.
5. Subir los archivos a GitHub.
6. Publicar en Netlify desde GitHub.
